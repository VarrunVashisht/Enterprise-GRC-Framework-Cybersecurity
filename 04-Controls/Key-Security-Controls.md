# Key Security Controls

## 1. Multi-Factor Authentication (MFA)
MFA reduces the risk of unauthorised access by requiring
multiple forms of verification.

- Addresses phishing and credential theft
- Required for critical systems
- Aligned with ISO 27001 access control requirements

## 2. Data Encryption
Encryption protects sensitive and personal data
from unauthorised disclosure.

- Applied to data at rest and in transit
- Supports Australian Privacy Act requirements

## 3. Regular Backups
Backups ensure data availability and recovery
in the event of ransomware or system failure.

- Backups are performed daily
- Restore tests are conducted periodically

## 4. Security Awareness Training
Training reduces human-related security risks.

- Conducted annually
- Covers phishing, passwords, and incident reporting

## 5. Incident Response Plan
Defines how the organisation responds to security incidents.

- Supports breach notification obligations
- Regularly reviewed and tested
