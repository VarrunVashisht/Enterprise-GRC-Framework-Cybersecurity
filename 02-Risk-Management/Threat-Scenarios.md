# Threat Scenarios

## 1. Phishing Attack Scenario
An attacker sends a phishing email impersonating a trusted source.
An employee clicks a malicious link and enters credentials.
The attacker gains unauthorised access to internal systems.

## 2. Ransomware Scenario
Malware is introduced via a malicious attachment.
Critical systems are encrypted, impacting operations and data availability.

## 3. Cloud Misconfiguration Scenario
Incorrect cloud security settings expose sensitive data to the internet.

## 4. Third-Party Breach Scenario
A vendor with system access suffers a data breach,
leading to indirect exposure of organisational data.
