# ISO 27001 Control Mapping

## Purpose
This document maps selected ISO/IEC 27001 Annex A controls
to organisational security controls.

## Control Mapping

| ISO 27001 Control | Description | Organisational Control | Evidence |
|------------------|-------------|------------------------|----------|
| A.5.1 | Information security policies | Information Security Policy | Policy document |
| A.8.2 | Information classification | Data classification rules | Data register |
| A.9.2 | User access management | MFA, Access reviews | Access logs |
| A.12.3 | Backup | Regular system backups | Backup reports |
| A.16.1 | Incident management | Incident Response Plan | Incident records |

## Review
This mapping is reviewed annually or upon changes to the ISO standard.
