# KPI and KRI Metrics

## Key Risk Indicators (KRIs)
KRIs provide early warning signals of increasing risk.

| KRI | Description |
|----|-------------|
| Number of critical risks | Risks rated as critical |
| Failed control tests | Controls not operating effectively |
| Security incidents | Number of reported incidents |
| Vendor risk issues | Open third-party risks |

## Key Performance Indicators (KPIs)
KPIs measure the effectiveness of GRC activities.

| KPI | Description |
|----|-------------|
| % controls tested on time | Control testing completion |
| % audit findings closed | Issue remediation rate |
| Compliance status | Regulatory compliance level |
| Incident response time | Speed of response |

## Review
Metrics are reviewed monthly and reported to management.
